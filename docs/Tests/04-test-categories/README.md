# Test Categories

This directory contains documentation for each test category in the AM-QADF testing framework.

## Categories

1. **[Unit Tests](unit-tests.md)** - Fast, isolated tests for individual components
2. **[Integration Tests](integration-tests.md)** - Tests for module interactions
3. **[Performance Tests](performance-tests.md)** - Benchmarks and regression tests
4. **[Property-Based Tests](property-based-tests.md)** - Mathematical property validation
5. **[E2E Tests](e2e-tests.md)** - Complete workflow tests

## Quick Reference

| Category | Files | Purpose | Speed |
|----------|-------|---------|-------|
| Unit | 141+ | Test individual functions/classes | < 1s |
| Integration | 6 | Test module interactions | 1-10s |
| Performance | 6 | Validate performance | Variable |
| Property-Based | 4 | Test mathematical properties | Variable |
| E2E | 3 | Test complete workflows | 10-60s |
