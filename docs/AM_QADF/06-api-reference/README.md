# API Reference

This directory contains complete API reference documentation for all AM-QADF modules.

## Module APIs

- **[Core API](core-api.md)** - Domain entities, value objects, and exceptions
- **[Query API](query-api.md)** - Data warehouse query clients
- **[Voxelization API](voxelization-api.md)** - Voxel grid operations
- **[Signal Mapping API](signal-mapping-api.md)** ⭐ - Interpolation methods (CRITICAL)
- **[Synchronization API](synchronization-api.md)** - Temporal/spatial alignment
- **[Correction API](correction-api.md)** - Geometric distortion correction
- **[Processing API](processing-api.md)** - Signal processing and noise reduction
- **[Fusion API](fusion-api.md)** - Multi-source data fusion
- **[Quality API](quality-api.md)** - Quality assessment
- **[Validation API](validation-api.md)** - Validation and benchmarking
- **[Analytics API](analytics-api.md)** - Statistical and sensitivity analysis
- **[Process Analysis: Prediction & Optimization API](process-analysis-prediction-api.md)** - Early defect prediction, time-series forecasting, model tracking, and enhanced optimization APIs
- **[SPC API](spc-api.md)** - Statistical Process Control (control charts, capability, multivariate SPC)
- **[Streaming API](streaming-api.md)** - Real-time data streaming (Kafka, incremental processing)
- **[Monitoring API](monitoring-api.md)** - Real-time monitoring (alerts, health, notifications)
- **[Deployment API](deployment-api.md)** - Production deployment utilities (configuration, scalability, fault tolerance, resource monitoring, performance tuning)
- **[Integration API](integration-api.md)** - Industrial system integration (MPM, manufacturing equipment, API gateway, authentication)
- **[Anomaly Detection API](anomaly-detection-api.md)** - Anomaly detection
- **[Visualization API](visualization-api.md)** - 3D visualization
- **[Voxel Domain API](voxel-domain-api.md)** - Main orchestrator

## Documentation Format

Each API reference includes:
- Class/function signatures
- Parameters and return types
- Usage examples
- Related classes/functions
- Cross-references to other APIs
