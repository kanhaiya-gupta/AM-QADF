# Usage Examples

This directory contains example code and tutorials for using AM-QADF.

## Example Categories

- **Basic Usage** - Simple examples for getting started
- **Signal Mapping** - Interpolation examples
- **Data Fusion** - Multi-source fusion examples
- **Quality Assessment** - Quality analysis examples
- **Analytics** - Statistical and sensitivity analysis examples
- **Anomaly Detection** - Anomaly detection examples
- **Visualization** - 3D visualization examples
- **Complete Workflows** - End-to-end examples

## Running Examples

```bash
# Run example script
python examples/basic_usage.py

# Run Jupyter notebook
jupyter notebook examples/notebooks/signal_mapping.ipynb
```
