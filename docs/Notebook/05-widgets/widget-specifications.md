## Quick Reference

### Standard Widget Layout

All notebooks follow a 5-panel layout:

- **Top Panel**: Mode selectors, action buttons
- **Left Panel**: Configuration accordion
- **Center Panel**: Visualizations and results
- **Right Panel**: Results display, statistics, export
- **Bottom Panel**: Status, progress, logs

### Common Widget Types

- **Dropdown**: Mode/method selection
- **RadioButtons**: Single choice
- **Checkbox**: Boolean toggles
- **Slider**: Parameter adjustment
- **Button**: Action triggers
- **Accordion**: Collapsible sections
- **Output**: Display areas
