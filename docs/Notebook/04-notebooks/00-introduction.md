## Purpose

This notebook provides an introduction to the AM-QADF framework, covering key concepts, architecture, and module overview. It serves as the starting point for learning the framework.

## Learning Objectives

By the end of this notebook, you will:

- ✅ Understand the AM-QADF framework purpose and goals
- ✅ Learn key concepts and terminology

- ✅ Understand the framework architecture
- ✅ Learn about data structures and workflows

## Topics Covered

### Framework Overview

- **Purpose**: Additive Manufacturing Quality Assessment and Data Fusion
- **Goals**: Multi-source data fusion, quality assessment, analytics
- **Applications**: Research, production, quality control

### Key Concepts

- **Voxel Domain**: 3D spatial representation
- **Signal Mapping**: Mapping point cloud data to voxel grids
- **Data Fusion**: Combining multi-source data
- **Quality Assessment**: Evaluating data quality
- **Analytics**: Statistical and sensitivity analysis

### Framework Modules

- **Core**: Domain entities and value objects
- **Query**: Data warehouse query clients
- **Voxelization**: Voxel grid operations
- **Signal Mapping**: Signal interpolation methods
- **Synchronization**: Temporal/spatial alignment
- **Correction**: Geometric distortion correction
- **Processing**: Signal processing
- **Fusion**: Multi-source data fusion
- **Quality**: Quality assessment
- **Analytics**: Statistical and sensitivity analysis
- **Anomaly Detection**: Anomaly detection methods
- **Visualization**: 3D visualization
- **Voxel Domain**: Main orchestrator
