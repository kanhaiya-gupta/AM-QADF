## Documentation Format

Each notebook documentation page includes:

1. **Purpose**: What the notebook teaches
2. **Learning Objectives**: What you'll learn
3. **Duration**: Estimated time to complete
4. **Topics**: Main topics covered
5. **Widgets**: Interactive widgets used
6. **Usage**: How to use the notebook
7. **Examples**: Example workflows
8. **Related**: Related notebooks and resources
